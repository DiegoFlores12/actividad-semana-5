# actividad-semana-5
Etapa 1: Desarrollo del archivo principal (index.html)

En esta etapa se implementó la estructura base de la página web, donde se integran los elementos que serán manipulados mediante JavaScript.

Contenidos principales:
Encabezado con el título de la actividad.
Párrafo dinámico para mostrar mensajes.
Texto informativo adicional (“Unidad 2”).
Contenedor <div> para visualizar contenido dinámico.
Botón interactivo para ejecutar funciones JavaScript.

Características técnicas:
Uso de etiquetas HTML básicas (h1, p, div, button).
Asignación de identificadores (id) para manipulación mediante DOM.
Integración de un script interno para mostrar un mensaje inicial en el div.
Enlace a archivo JavaScript externo (actividad.js).


Etapa 2: Implementación del archivo JavaScript (actividad.js)

Se desarrolló un archivo externo que contiene la lógica de interacción con la página.

Funcionalidad:
Definición de la función mostrarMensajes(), la cual se ejecuta al presionar el botón.
Manipulación de elementos HTML mediante getElementById.

Acciones implementadas:
Mostrar una alerta emergente con alert().
Modificar el contenido de un párrafo (<p>).
Mostrar mensajes en la consola con console.log().
Actualizar el contenido de un <div>.
 
Etapa 3: Ejecución de mensajes en el navegador

Se implementaron cuatro formas distintas de visualizar información.

Métodos utilizados:
Alerta emergente → interacción directa con el usuario.
Mensaje en párrafo → visualización dentro del documento.
Consola → salida de depuración.
Mensaje en <div> → contenido dinámico en contenedor.

Cada método cumple un propósito distinto dentro del desarrollo web.
 
Etapa 4: Estructura del proyecto

El proyecto se organizó de forma simple para facilitar su comprensión.

Archivos utilizados:
index.html → estructura principal
actividad.js → lógica del programa

Organización:
Separación entre estructura (HTML) y funcionalidad (JavaScript).
Código limpio y fácil de mantener.
