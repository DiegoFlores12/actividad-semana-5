# actividad-semana-5
Objetivo

Comprender y aplicar distintas formas de mostrar mensajes en JavaScript dentro de una página web, utilizando alerta, DOM, consola y un contenedor <div>.

Cómo ejecutar:
Abrir el archivo index.html en un navegador (Chrome, Edge o Firefox).
Se mostrará el contenido de la página.
Presionar el botón “Ejecutar Mensajes”.
Verificar los resultados:
alerta emergente
mensaje en la página
salida en consola (F12)
mensaje en el div
